Print and Wet Sign Upload Guidance:
  • Recommended Paper Size:
  • A4 – When the number of exhibit columns is less than 8
  • A3 – when the number of exhibit columns exceed 8
  • Note – In the event the number of columns exceed 25, Admin user may have to overwrite the exception to proceed with wet sig upload
  •  
  • Recommended Scanner Setting:
  • Scanner resolution must be set to at least 300dpi
  •  
  • Recommended Page Orientation for Exhibit:
  • Landscape
